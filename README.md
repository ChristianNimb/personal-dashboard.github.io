# Personal Dashboard 🚀

A beautiful, customizable browser start page with weather, tasks, quick links, and inspirational quotes.

![Dashboard Preview](https://img.shields.io/badge/Status-Live-success?style=for-the-badge)
![License](https://img.shields.io/badge/License-MIT-blue?style=for-the-badge)

## ✨ Features

- **⏰ Live Clock** - Real-time clock with date display
- **🌤️ Weather Widget** - Live weather using wttr.in API (customizable location)
- **🔍 Google Search** - Quick search bar (Google or direct URL)
- **👋 Personal Greeting** - Time-based greetings with your name
- **✅ Todo List** - Full task management with persistence
- **🔗 Quick Links** - Customizable favorite sites with emoji icons
- **💭 Inspirational Quotes** - Rotating developer quotes
- **🎨 Themes** - 6 beautiful gradient themes + custom backgrounds
- **🖼️ Custom Backgrounds** - img upload or random Unsplash images
- **💾 Data Persistence** - Everything saves to localStorage

## 📦 Installation

### Use as Browser Start Page

1. **Download** the `index.html`,`styles.css` and `web.js` file
2. **Open** your browser settings:
   - **Chrome:** Settings → On startup → Open a specific page
   - **Firefox:** Settings → Home → Custom URLs
   - **Edge:** Settings → Start, home, and new tabs
3. **Set** the path to your downloaded `index.html` file

### For New Tab Page

Use a browser extension like:
- Chrome: "Custom New Tab URL"
- Firefox: "New Tab Override"

Then point it to your `index.html` file location.

## 💡 Usage

### Customization

1. **Click the settings gear** (⚙️) in the bottom right
2. **Choose a theme** or upload custom background
3. **Set your location** for weather
4. **Add your name** for personalized greetings
5. **Add quick links** by clicking the + button

### User Flow:

1. **Click settings** ⚙️
2. **Click** "🖼️ Custom Background"
3. **Click** "Choose File"
4. **Select image from computer**
5. **Click** "Upload Image"
6. ✓ **Background changes instantly!**
7. **Close settings**
8. **Enjoy your custom background**
### Features

- **Search:** Type in the top bar, press Enter
- **Todo:** Add tasks, check them off, delete when done
- **Weather:** Click to open settings and change location
- **Quotes:** Click "New Quote" for fresh inspiration
- **Links:** Click + to add, click link to visit

## 🔧 Customization

All data is stored in `localStorage`:
- `theme` - Selected theme
- `customBg` - Custom background URL
- `weatherCity` - Weather location
- `userName` - Your name
- `links` - Quick links array
- `todos` - Tasks array
